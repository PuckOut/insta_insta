    InstaScrapex

    Scrapes Instagram JSON and HTML endpoints without and API key

    Use at your own risk

    Installation

    1. Add insta_insta to your list of dependencies in mix.exs:

      def deps do
  [{:insta_insta, "~> 0.0.1"}]
end

    2. Ensure insta_insta is started before your application:

      def application do
  [applications: [:insta_insta]]
end

    Usage

    All functions return a map in the response[:body] Functions wrap HTTPoison.get!/3 so you can pass headers and other options to them
